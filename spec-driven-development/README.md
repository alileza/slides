# Spec-Driven Development Presentation

A 15-20 minute presentation about my journey from AI skeptic to advocate through spec-driven development.

## Overview

This presentation tells a personal story of discovering how to make AI coding assistants reliable through structure and specificity, featuring:

- **Act 1: The Skeptic** - Late 2024 doubts about AI coding
- **Act 2: The Explorer** - Trial and error with different approaches
- **Act 3: The Discovery** - Finding spec-driven development
- **Act 4: The Insight** - Core principles over frameworks
- **Act 5: The Vision** - Executable specs as the future

## Running the Presentation

### Install dependencies
```bash
pnpm install
```

### Start the server
```bash
pnpm dev
```

Then open http://localhost:8000/presentation.html

## Key Themes

- Specificity vs vagueness in AI instructions
- Four core principles: Chaining, Isolated Context, Documentation, Context Compaction
- Comparison of frameworks: SpecKit, BMAD, OpenSpec, Agent OS
- Future vision: BDD for product building with executable specs
